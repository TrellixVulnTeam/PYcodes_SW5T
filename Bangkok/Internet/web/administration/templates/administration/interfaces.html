Login

POST
http://pelocano.herokuapp.com/service/login/
Mensagem:
{ "username": "", "password": "" }
Resultado:
{ "success": true } ou { "msg": "username and password do not match!!", "success": false }

Registrar Jogador

POST
http://pelocano.herokuapp.com/service/user_registration/

Mensagem:
{ "username": "", "email": "", "password": "", "name": "", "schooling": "", "gender": "", "age": 0 }
Resultado:
{ "success": true }
